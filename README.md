# Developer System Portfolio

An interactive, game-inspired developer portfolio built using **HTML, CSS, and JavaScript**.

Instead of a traditional scrolling portfolio, this project presents the recruiter with a **Developer Evaluation System**, where they progress through multiple evaluation phases to explore the candidate's profile, skills, projects, achievements, and contact information.

---

## Features

* Terminal-style boot sequence with typing animation
* Interactive evaluation-based navigation
* Five structured evaluation phases
* Candidate overview
* Technical skills analysis
* Project archive
* Achievement records
* Communication terminal
* Final evaluation completion screen
* Smooth screen transitions and animations
* Responsive and futuristic user interface

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## Evaluation Phases

### Phase 01 – Candidate Overview

Introduces my profile, educational background, current skills, and career objective.

### Phase 02 – Technical Analysis

Displays my programming languages, web development skills, and technologies I am currently learning.

### Phase 03 – Project Archive

Showcases my major projects, including:

* Developer System Portfolio
* Hotel Management Website

### Phase 04 – Achievement Records

Highlights coding competition participation and professional certifications.

### Phase 05 – Communication Terminal

Provides access to my email, GitHub, and LinkedIn profiles.

---

## Project Inspiration

This portfolio is inspired by game-style mission and evaluation systems. Instead of navigating through a conventional portfolio website, visitors progress through a structured developer evaluation, creating a more engaging and memorable experience while maintaining a professional presentation.
